Detailed Description

Region class stores boundaries of a region, a vector of bodies in the region, a nodepath and a boolean that checks if collisions have been computed.

Parameters
regnodepathThis is a string that encodes the path one takes to reach some child node from the root node of the tree.
checkcolThis checks if collision has been computed. This is set to false initially, and set to true if collisions are checked, preventing needless extra computations at child nodes.
